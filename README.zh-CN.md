# 在 Vercel 上部署 ChatFlow

要求：

- 使用 ChatFlow 作为模板
- 注册 Vercel 帐户

## 使用

1. 登录到您的 Vercel 帐户并单击 “New Project”。
2. 选择您的 Git 代码仓库，其中存储着您的 Next.js 应用程序。
3. Vercel 将自动检测到它是一个 Next.js 应用程序，并配置构建设置。
